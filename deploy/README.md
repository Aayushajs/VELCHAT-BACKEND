# deploy/ — GitOps config (no secrets)

| Path                      | What                                                                                                                            |
| ------------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| `helm/velchat-service/`   | Generic chart for any stateless service (Deployment + Service + HPA + PDB + anti-affinity + probes + non-root securityContext). |
| `helm/values/<svc>.yaml`  | Per-service overrides (image, ports, envFrom). Generated by `tools/scaffold`.                                                   |
| `argocd/app-of-apps.yaml` | ApplicationSet that deploys all 13 services from the generic chart.                                                             |
| `k8s/base/`               | Namespace + default-deny NetworkPolicy baseline.                                                                                |

**Secrets never live here.** ConfigMaps hold non-secret env; secrets come from Sealed Secrets / Vault (`envFrom.secretRef`), installed in **P0**.

P0 expands this with: data-tier operators (CloudNativePG, Mongo, Valkey, Strimzi/Kafka, OpenSearch, MinIO), Linkerd mTLS, cert-manager, and the observability stack (Prometheus/Grafana/Loki/Tempo/GlitchTip).

Render a service locally:

```bash
helm template auth-service deploy/helm/velchat-service -f deploy/helm/values/auth-service.yaml
```
